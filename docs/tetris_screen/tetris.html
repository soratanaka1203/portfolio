なぜこのゲームを作ったか

このゲームを製作した理由は、パズルゲームを作ってみたかったからです。
また、バンタンの授業で2次元配列について習ったときにシミュレーションゲームやパズルゲームなどに
利用できると聞いたので試してみたいと思いこのゲームを製作しました。

こだわったところ

このゲームで私がこだわったところは、２次元配列を活用しブロックの管理を行ったところです。
特に、ブロックを回転させる処理は行列の転置を使って実装することができたので良かったです。

実際のコードです↓

         
            // ▼ テトロミノの右回転（行列の転置＋行の反転を応用）
            function rightRotate() {
                let nTetoro = [];
                for (let y = 0; y < tetoro_size; y++) {
                    nTetoro[y] = [];
                    for (let x = 0; x < tetoro_size; x++) {
                        // 転置（y,x → x,y）＋ 行の反転（tetoro_size - x - 1）
                        nTetoro[y][x] = tetoro.shape[tetoro_size - x - 1][y];
                    }
                }
                return { shape: nTetoro, color: tetoro.color };
            }
